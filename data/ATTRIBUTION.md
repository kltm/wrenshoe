# Data Attribution

This directory contains flashcard decks. Each deck JSON carries its own
`license` (SPDX identifier) and `sources` (upstream attribution) fields.
This file provides a human-readable summary.

  - URL: https://github.com/jamsinclair/open-anki-jlpt-decks
